The files in this folder are used to help choose scattering time parameters that are consistent with existing experiments.

kappa_GaAs_fit.m:  uses experimental k vs. T  to obtain Umklapp parameters for GaAs
kappa_InAs_fit.m:  used experimental k vs. T to obtain Umklapp parameters for InAs
kappa_InGa_fit.m:  uses experimental k vs. x to obtain alloys scattering parameter, F.
kappa_ErAsInGaAs_vs_VolFrac: used to plot k vs. nanoparticle volume fraction for a particular nanoparticle (not used in manuscript)
